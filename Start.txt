Для запуска проекта
- yarn install
- yarn start

База mongoDB на стандартном порту без пароля
Установка данных: http://localhost:3000/setup-db

Стартовая странрица и страница для входа: http://localhost:3000/
Аутентификация базируется на jwt, перед запросом к данным проверяется ключ, если его нет данные не приходят для таблицы

Страница регистрации нового юзера: http://localhost:3000/register

При скроле данные подгружаются кусками, чтобы не нагружать сразу фронт и бек
При сортровке возвращается новый стейт из редакса
Поиск сделан по имени
